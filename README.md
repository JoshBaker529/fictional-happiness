# fictional-happiness
Emulator of the future!

This is for my learning experience, written in C++.

## Directory Layout
### /src/
This holds all of the source files

### /cmd/
This is where any and all executables are produced, along with any data files required.

## How to use
Running the command `make` will produce the executable `/cmd/term/term.out`.
This can then be run to launch the application.
